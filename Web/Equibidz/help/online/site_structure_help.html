Administrator Help - Web Site Structure and Design Structure

Below is a map of the Visual Auction 3.0 directory structure.
 Pink files can modified to personalize your info and graphics.
Visual Auction 3.0:

admin:

  • This section contains the Administration pages.  These pages can only be modified by Beyond Solutions, Inc.

bid:

  • Pages in this section process the user's bid.  No changes need to be made to these pages.

buyers:

  • index.html: This file can be changed to add more information or graphics.

dbase:

  • This directory contains the auction database. There are no modifiable files in this directory. 

event1:

  • listings:
    • categories:
      • gen_categories.cfm, default_h.cfm, default_f.cfm: These files generate all /listings/categories and HTML files.  The default_h.cfm (default header) and the default_f.cfm (default footer) files will be included as the header and footer of each generated file.  Custom header and footer files for each category can be included by placing header and footer files in this directory, where the name of those custom files will be: (category number)_h.cfm and (category number)_f.cfm.  Both header and footer files must exist to be included in gen_categories.cfm file.
    • featured:
      • gen_featured.cfm: This file generates all /listings/featured/*.* files.
    • primary_listings:
      • gen_listings.cfm: This file generates all files in these directories:
        1. /listings/completed/*.*
        2. /listings/current/*.*
        3. /listings/ending/*.*
        4. /listings/going/*.*
        5. /listings/new/*.*
  • pages:
    • root:
      • gen_index.cfm: This file generates your main index.html page accessable at http://www.yourdomain.com/

event2:

  • This directory contains files which run on a schedule.  Their function is to complete background processes, including billing, invoicing, future-watch, and proxy bidding.  No files in this directory should be modified.

feedback:

  • This directory contains files which are generated 'on the fly'.  No changes need to be made directly to these pages.

help:

  • index.html: This file can be changed to add more information or graphics.
  • sitemap.html: This file can be changed to add more information or graphics.
  • auctiontypes.html: This file can be changed to add more information or graphics.
  • fee_schedule.html: This file can be changed to add more information or graphics.
  • admin: This directory contains help files for the auction Administrator.  No changes need to be made.

images:

  • logohere.gif: This image should be replaced with your company logo. (The image name should not be changed.)
  •  infohere.jpg: This image should be replaced with an image providing site info or an image map. (The image name should not be changed.)

includes:

  • menu_bar.cfm: This file can be modified to personalize your menu bar site-wide. You may also cut and paste this file to appear at the top of any HTML page needing the menu bar. No other files in this directory should be modified!

listings: The following directories contain automatically generated files relating to auction item listings. These do not need to be modified.

  • categories: This directory contains files relating to categories.
  • completed: This directory contains files for completed auctions.
  • current: This directory contains files for current auctions.
  • details: This directory contains files for auction details.
  • ending: This directory contains files for auctions ending.
  • featured: This directory contains files for featured auctions.
  • going: This directory contains files for going auctions.
  • new: This directory contains files for new auctions.

messaging: This directory contains files relating to messaging. No need to modify.

  •  

personal: This directory contains files relating to personal info. No need to modify.

  •  

registration: This directory contains files relating to user registration. No need to modify.

  •  

search: This directory contains files relating to the site search engine. No need to modify.

  •  

sell: This directory contains files relating to selling issues. No need to modify.

  •  

sellers:

  • index.html: This file can be changed to add more information or graphics.
Remember ...
  • All generated pages will have a time stamp like this: "last updated: mm/dd/yyy, hh:mm:ss".  These pages are scheduled to re-generate every 2 hours.   In order to make changes to pages displaying this stamp, you will need to modify their respective source file.
